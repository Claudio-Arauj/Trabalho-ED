# ED
 Trabalho de Estrutura de Dados para o relatório de busca de dados
 
# Como Fazer
 Para gerar os resultados, basta compilar por um compilador compatível com a linguagem C, o compilador usado para o relatório foi o gcc.
 usando gcc de exemplo basta colocar:
 gcc -o nome_do_arquivo nome_do_arquivo.c
 
 Após repetir isso com todos os arquivos, isso vai gerar o executável de cada um, no qual deve-se rodar na maquina e esperar os arquivos .txt aparecerem.
 Assim ao fazer isso com todos os .txt gerados, execute os arquivos do gnuplot usando o terminal do computador colocando o seguinte comando como exemplo: 
 gnuplot 'exemplo_script'.gp
 
 Com isso, ao rodar todos os sripts.gp vai se ter todos os graficos gerados com o tempo de execução e o numero de arrays.
